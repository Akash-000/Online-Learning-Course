About This Course


As the first course in the Android Developer Nanodegree, Developing Android Apps is the foundation of our advanced Android Curriculum. This course blends theory and practice to help you build great apps the right way. In this course, you'll work with instructors step-by-step to build Android apps and learn best practice of mobile development and Android development in particular.

Shareable Certificate


What will you learn ?

1

Android and Java Foundations

Get started with Android and Java
2

Wireframing, Java and Android Studio

Wireframe your first app, learn about basic concepts in Java and dig into the Android Studio IDE
3

Android App Development

Really dig into Android development fundamentals by working with Android Layouts & Views and handling user interactions
4

Intermediate Java and Capstone Project

Pick up more Java programming skills and complete your OWN application

What projects will you build ?



Bug Invaders

Re-create an arcade classic: Space Invader !



Treasure Hunter

Dig for the treasure by looping through various arrays !



Business Outfitted Alice

Create an outfit simulation game
